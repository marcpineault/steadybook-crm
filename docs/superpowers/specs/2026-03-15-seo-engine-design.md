# Phase 2: SEO Engine — Design Spec

## Overview

Add SEO capabilities to the autonomous nightly system. Claude Code audits calmmoney.ca for technical SEO issues, researches keywords, analyzes competitors, and generates blog content drafts — all automatically during the nightly run.

**Goal:** Dominate "financial advisor London Ontario" and related long-tail keywords on Google.

## Architecture

The SEO engine is NOT a new module in the bot codebase. It's a set of tasks added to `AUTONOMOUS.md` that the nightly Claude Code run executes against BOTH repos:

1. **calm-money-bot** (this repo) — SEO research saved to `content/seo/`
2. **Pineault-wealth** (`/Users/map98/Desktop/Pineault-wealth/`) — the Next.js website where technical SEO fixes and blog content are committed

## What Gets Added

### 1. SEO Research Tasks in AUTONOMOUS.md

Add to the Weekly section:

```markdown
## Weekly (Sunday only)
- Research SEO for calmmoney.ca:
  - Check current Google ranking signals (meta tags, structured data, page speed indicators)
  - Research 5-10 long-tail keywords for London Ontario financial planning
  - Analyze top 3 competitor websites for content gaps
  - Save findings to content/seo/YYYY-MM-DD-research.md
- Generate 1 blog post draft based on highest-opportunity keyword — save to content/seo/drafts/
```

### 2. Technical SEO Audit Tasks

Add to the Every Run section (applied to Pineault-wealth repo):

```markdown
## Every Run
- In /Users/map98/Desktop/Pineault-wealth/:
  - Check all pages have proper meta titles and descriptions
  - Verify structured data (JSON-LD) is present for LocalBusiness schema
  - Check for broken internal links
  - Verify sitemap.xml is up to date
  - Check image alt tags are present and descriptive
  - Fix any issues found, commit to Pineault-wealth repo
```

### 5. Blog Content Strategy

Each blog post targets one long-tail keyword and follows this structure:
- Title with keyword naturally included
- 800-1500 words, plain language, Marc's voice
- Internal links to calmmoney.ca services pages
- CTA pointing to quiz or booking widget
- Proper meta description and title tag
- Generated by Claude Code during nightly run, saved as draft for Marc to review

## Implementation

This is primarily about updating `AUTONOMOUS.md`, `run-autonomous.sh`, and creating the directory structure. No new Python modules needed — the nightly Claude Code session does the actual SEO work using its web search and file editing capabilities.

## Constraints

- Blog drafts are saved as files, not auto-published — Marc reviews before posting
- Technical SEO fixes to Pineault-wealth are committed and pushed automatically (tests permitting)
- No paid SEO tools — relies on Claude Code's web research and code analysis
- Keyword tracking is manual (saved in markdown) — no Google Search Console API integration yet
