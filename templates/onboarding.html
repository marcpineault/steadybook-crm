Step 1 of 7
Dashboard API Key
This is the secret password that protects your dashboard. Generate one now — you'll paste it into Railway in the final step.
⚠ Save this somewhere safe before continuing.
Step 2 of 7
Telegram Bot Token
This lets the CRM send you alerts on Telegram when new leads come in or tasks need your attention.
  1. Open Telegram and search for @BotFather
  2. Send the message: /newbot
  3. When asked, type a display name for your bot (e.g. "My CRM Bot")
  4. When asked, type a username ending in "bot" (e.g. "mycrm_notifications_bot")
  5. BotFather will reply with a token — it looks like: 123456789:ABC-DEF1234ghijk
  6. Copy that token and paste it below
Paste exactly as given by BotFather — no spaces.
Step 3 of 7
Your Telegram Chat ID
This tells the bot which Telegram account to send alerts to. We'll send you a test message to confirm.
  1. In Telegram, search for @userinfobot and start a chat
  2. Send it any message — it will reply with your user info
  3. Copy the Id number (e.g. 123456789)
  4. Then go find your own bot (the one you just created) and send it any message — this is required before it can message you back
  5. Paste your ID below and click "Send Test Message"
Numbers only, no spaces or symbols.
Step 4 of 7
OpenAI API Key
Powers all the AI features — writing follow-up messages, summarising conversations, scoring leads, and more.
  1. Go to platform.openai.com/api-keys
  2. Sign in or create an account
  3. Click Create new secret key, give it a name like "SteadyBook"
  4. Copy the key — it starts with sk-
  5. Make sure your account has billing set up (Settings → Billing → Add payment method)
Step 5 of 7
Email Sending (Resend)
Used to send automated follow-up emails and nurture sequences to your leads.
  1. Go to resend.com and create a free account
  2. Go to Domains and add your domain (e.g. yourdomain.com) — follow the DNS verification steps
  3. Go to API Keys and click Create API Key
  4. Copy the key (starts with re_) and paste it below
  5. Set your "From" email to something like hello@yourdomain.com — must match the domain you verified
Emails will be sent from this address. Must match your verified Resend domain.
Step 6 of 7
SMS Sending (Twilio) Optional
Lets the CRM send and receive text messages with your leads. Skip this if you don't need SMS right now.
  1. Go to twilio.com and create an account
  2. From your Console dashboard, find your Account SID and Auth Token
  3. Go to Phone Numbers → Manage → Buy a number and get a number with SMS capability
  4. Paste the SID, Token, and phone number below
  5. Phone number format must include country code: +14165551234
Include the + and country code. Example: +14165551234
Step 7 of 7
n8n Webhook Secret Optional
If you plan to capture leads from Instagram, LinkedIn, or WhatsApp using n8n, this secret key keeps that connection secure. Skip if you're not using n8n yet.
  1. n8n is a free automation tool — it watches your social accounts and forwards new leads to SteadyBook automatically
  2. When someone DMs you on Instagram or fills in a LinkedIn lead ad, n8n captures their info and sends it here
  3. The webhook secret is like a password — it proves the data really came from your n8n, not someone else
  4. Generate a secret below, then paste it into your n8n Webhook node under Header Auth, with header name X-Webhook-Secret
  5. Your SteadyBook webhook URL will be: https://[your-app].railway.app/api/social/intake
